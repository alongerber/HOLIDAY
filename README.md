# קפריסין · בונה תקציב חופשה

כלי ניהול פרויקט חופשה: תקציב דינמי, השוואת טיסות ווילות, צ׳ק-ליסט משימות, הפקת דוח PDF,
ריבוי פרויקטים, וקליטת עלויות אוטומטית מצילומי מסך באמצעות Claude.

## מבנה
```
index.html      – כל האפליקציה (פרונט, ללא build)
api/parse.js     – פונקציית שרת: מקבלת צילום, שולחת ל-Claude, מחזירה JSON מובנה
vercel.json      – הגדרות פריסה
```

## פריסה ל-Vercel (5 דקות)
1. צור repo ב-GitHub עם הקבצים האלה (או גרור את התיקייה ל-Vercel > New Project).
2. ב-Vercel, ב-**Settings → Environment Variables**, הוסף:
   - `ANTHROPIC_API_KEY` = מפתח ה-API שלך מ-console.anthropic.com
3. Deploy. זהו.

## מה עובד בלי backend (גם מקומית / כל אירוח סטטי)
- כל התקציב, הטיסות, הווילות, המשימות, הדוח, ריבוי הפרויקטים — נשמר ב-localStorage בדפדפן.
- שאיבת תמונת נכס מקישור (דרך microlink) — עובד בדפדפן.

## מה דורש את ה-backend (Vercel + מפתח API)
- **קליטת AI מצילום** — העלאת צילום מסך → הפונקציה `api/parse.js` קוראת ל-Claude vision,
  מזהה סוג עלות / ספק / סכום / מטבע / תאריך, והאפליקציה מציעה להוסיף כפריט או לעדכן שדה.
  בלי המפתח, האפליקציה תציג "לא זמין" ותפנה לחלץ ידנית.

## ריבוי פרויקטים
בסרגל העליון: כל בן משפחה פותח "פרויקט חדש" ומקים תקציב משלו מאפס (נשמר בנפרד).
`שכפל` מעתיק פרויקט קיים כבסיס; `מחק` מוחק את הנוכחי.

## הערות
- ה-`api/parse.js` משתמש במודל `claude-sonnet-5` — אפשר להחליף למחרוזת מודל אחרת בקובץ.
- מגבלת גוף בקשה ב-Vercel היא ~4.5MB; צילומי מסך רגילים הרבה מתחת לזה.
- כל הערכי "הערכה" בכלי הם ברירת מחדל — עדכן מול הצעות אמיתיות.
